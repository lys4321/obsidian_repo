**Crystal Reports**
- 개발사 : Seagate Software -> SAP 
- 라이선스 유형 : 상용	
- 사용 분야 : 기업, 공공기관
- 비용
	- SAP Crystal Reports(단일 사용자 라이선스) : $495 (1회 구매)
	- SAP Crystal Server (NUL/CAL) : $869 ~ $8,744 (사용자 수/동시 사용자 수 기반)
- 특징
	SAP가 중소기업을 대상으로 마케팅하는 비즈니스 인텔리전스 애플리케이션이며,
	다양한 데이터 소스를 기반으로 정형화된 보고서를 생성하는 데 특화된 기업용 리포팅 도구
- 장점
	- 다양한 데이터 소스(DB, Excel, Google Sheet 등) 연결 지원
	- 표, 차트, 그래프 등 데이터 표현 방법을 지원
	- 다양한 배포/출력 형식(PDF, Excel, Word 등)을 지원
- 단점
	- 많은 기능을 지원하는 만큼 이에 대한 교육이 필요
	- 최신 버전인 크리스탈 리포트 2020도 2026년 말 지원


**Jaspersoft Report**
- 개발사 : TIBCO Software
- 라이선스 유형 : 공용/상용
- 사용 분야 : 기업, 개발자
- 비용
	
- 특징
	Jaspersoft Studio를 이용해 보고서 템플릿을 작성하고
	JasperReports Library + JasperReports Server를 이용해 보고서를 생성, 출력하는
	Java 기반의 오픈 소스 보고서 생성 도구
- 장점
	- Java 환경에서 뛰어난 확장성
	- 클라우드, 온프레미스 환경에 대한 배포방법 지원
	- BI기능 지원
	- Docker, AWS, Azure 등과 같은 다양한 환경에서 배치 가능
- 단점
	- 고급 기능에 대한 학습이 필요
	- BI 기능이 최신 툴 대비 좋지 않음
	- 한국어 및 국내 환경 지원 아쉬움

**Tableau**
- 개발사 :  Tableau Software -> Salesforce
- 라이선스 유형 : 상용
- 사용 분야 : 기업의 BI
- 비용
	- Creator (데이터 모델링, 대시보드 제작) : $75/월/사용자 (연간 청구)
	- Explorer (자체 분석, 보고서 수정) : $42/월/사용자 (연간 청구)
	- Viewer (대시보드 열람) : $15/월/사용자 (연간 청구)
- 특징
	데이터를 시각적으로 분석하고 공유할 수 있는 대표적인 BI 도구
- 장점
	- 표, 차트, 그래프 등 데이터 표현 방법을 지원
	- 대시보드 모니터링 및 보고서 작성 기능을 지원
	- Excel, 데이터베이스, 클라우드, API 등 거의 모든 데이터와 연결을 지원
	- 대규모 데이터를 처리 가능하며 데이터 통합 및 혼합 기능 지원
- 단점
	- 기본적인 시각화 기능은 쉽지만 고급 분석이나 시각화 기능은 교육이 필요
	- 대용량이거나 복잡한 데이터셋을 처리 시 성능 저하가 발생할 수 있음

**OZ Report**
- 개발사 : Forcs
- 라이선스 유형 : 상용
- 사용 분야 : 공공기관, 금융권
- 비용

- 특징
	기업 보고서의 개발, 운영, 관리 및 직관적인 UI를 지원하는 엔터프라이즈 리포팅 솔루션
- 장점
	- 국내 대상으로 하여 한글 지원 및 공공기관/기업 시스템과의 연계가 용이
	- HTML5 뷰어를 제공하여 웹 브라우저 환경에서 보고서를 조회/출력
	- 정형 보고서 제작에 적합
	- 다양한 배포/출력 형식(PDF, Excel, HTML 등) 을 지원
- 단점
	- 'OZ Script' 라는 JS 기반의 스크립트 언어 사용하여 학습이 필요
	- 정형 보고서 특화 기능을 가져 시각화 및 실시간 데이터 분석 등 BI 기능은 제한
	- 초기 세팅 작업이 복잡
	- 레퍼런스 자료가 비교적 적음

**ClipReport**
- 개발사 : 한컴이노스트림
- 라이선스 유형 : 상용(구독형)
- 사용 분야 : 기업의 BI, 마케팅
- 비용

- 특징
	기업이나 기관의 여러 소스에 있는 데이터를 가져와 통계 양식, 증명서 등 각종 보고서로 만들고, 웹과 모바일 환경에서 조회 및 출력할 수 있도록 지원하는 소프트웨어
- 장점
	- 국내 대상으로 하여 한글 지원 및 공공기관/기업 시스템과의 연계가 용이
	- HTML5 뷰어를 제공하여 웹 브라우저 환경에서 보고서를 조회/출력
	- 정형 보고서 제작에 적합
	- 다양한 배포/출력 형식(PDF, Excel, HTML 등) 을 지원 및 자체 개발한 차트, 크로스탭, 바코드, 서브리포트 등의 컴포넌트를 제공
- 단점
	- 정형 보고서 특화 기능을 가져 시각화 및 실시간 데이터 분석 등 BI 기능은 제한
	- 초기 세팅 작업이 복잡
	- 레퍼런스 자료가 비교적 적음

**FineReport**
- 개발사 : FanRuan Software
- 라이선스 유형 : 상용
- 사용 분야 : 기업, 공공기관
- 비용

- 특징
	- 간단한 드래그 앤 드롭만으로 다양한 대시보드를 제작
	- 데이터 시각화 툴로서 뿐만 아니라 데이터 작성 & 의사결정 분석 시스템을 쉽게 구축
- 장점
	- 다양한 데이터 소스를 연결하고 결합하여 보고서를 생성/관리하는 기능 제공
	- 3D 차트, 지도, GIS 관계, 대시보드등과 같은 시각화 옵션 제공
	- 다양한 배포/출력 형식(PDF, Excel, HTML 등) 을 지원
	- 자체 개발한 데이터 입력(Data Entry) 기능을 제공하여 온라인 폼을 통해 실시간으로 데이터를 수집하고 데이터베이스에 입력
- 단점
	- 정형 보고서에 특화되어 OLAP 기능이 상대적으로 제한
	- 고급 기능에 대한 학습이 필요
	- 고성능 BI 대비 대용량 데이터 처리 속도가 느려질 수 있음
	- 중국에서 개발된 SW이기 때문에 보안·정책 이슈로 도입에 제약이 있을 수 있음

**Zoho Analytics**
- 개발사 : Zoho Corporation
- 라이선스 유형 : 상용
- 사용 분야 : 기업의 BI, 마케팅
- 비용
	- 엔터프라이즈(Enterprise) : $575/월 
		- 연단위 청구, 사용자: 50명, 행: 5천만 개
		- 사용자 및 행 추가 시 각각 추가 비용
- 특징
	다양한 데이터 소스를 통합하여 사용자가 직접 데이터를 분석하고 시각화된 대시보드를 만드는 데 도움을 주는 셀프서비스 비즈니스 인텔리전스(BI) 플랫폼
- 장점
	- Zoho 사의 다른 애플리케이션과 연동이 쉬움
	- Ask Zia라는 AI 기능을 통해 사용자의 자연어 쿼리를 기반으로 보고서와 KPI 위젯을 생성하고 시각화 추천을 제공하는 등 고급 분석 기능을 지원
	- 모바일 앱 버전도 있어서 모바일 환경 지원
- 단점
	- Zoho 사의 애플리케이션를 제외한 다른 경우 통합이 제한이 되거나 어려울 수 있음
	- 대용량이거나 복잡한 데이터셋을 처리 시 성능 저하가 발생할 수 있음
	- 정형 보고서 출력 기능이 제한
	- 한글 지원이 약함
	- 클라우드 환경에 최적화 되어 있어 자체 서버를 두기엔 어려움

**UbiForm**
- 개발사 :  유비스톰
- 라이선스 유형 : 상용
- 사용 분야 : 기업, 공공기관
- 비용

- 특징
	HTML5 기반의 전자 서식 및 웹 리포팅 솔루션
- 장점
	- 클라우드, 온프레미스 환경을 지원하는 웹 기반 솔루션
	- 국내 대상으로 하여 한글 지원 및 공공기관/기업 시스템과의 연계가 용이
	- 전자서명 기능이 내장되어 있어 공공기관/기업 시스템에 용이
	- 다양한 배포/출력 형식(PDF, Excel 등)을 지원
- 단점
	- 정형 보고서에 특화되어 BI 수준의 데이터 분석 기능은 제한
	- 자체 개발한 독자 기술 및 에디터에 의존하여 환경에 제약
	- 고급 기능에 대한 학습이 필요

**Power BI**
- 개발사 : Microsoft
- 라이선스 유형 : 공용/상용
- 사용 분야 : 기업, 공공기관
- 비용
	- Power BI Premium (Per Capacity) : $4,995/월부터
- 특징 
	비즈니스 분석 및 데이터 시각화 도구로, 엑셀과 비슷한 직관적인 인터페이스를 갖추고 있어 데이터 전문가가 아니더라도 쉽게 사용이 가능
- 장점
	- Microsoft 제품과의 뛰어난 통합
	- 다양한 데이터 소스(DB, Office, Azure 등) 연결 지원
	- AI를 사용하여 사용자의 자연어 질의, 주요 영향 요인 분석 등 다양한 AI 기능을 활용
	- 다양한 기본 시각화 개체와 앱 소스에서 추가할 수 있는 커스텀 시각화 옵션을 제공
- 단점
	- 고급 기능에 대한 학습이 필요
	- 대용량이거나 복잡한 데이터셋을 처리 시 성능 저하가 발생할 수 있음
	- Microsoft의 제품이 아닌 경우 호환이 어려울 수 있음
	- 클라우드 기반으로 작동하여 보고서 및 대시보드에 접근하려면 인터넷 연결이 필요
